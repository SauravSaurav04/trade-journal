About the App

Trade Tracker is a Spring Boot-powered web app that helps traders stay organized by recording every trade they make. With insightful analytics and a clean interface, it supports smarter trading decisions and better performance tracking.

Key Features

📈 Track Trades

Log each trade with details like entry, exit, instrument, and strategy notes.

📊 Analytics

Visualize profit/loss trends, win rates, and trade performance over time.

📝 Trade Notes

Reflect on your decisions by adding notes and thoughts to each trade.

🔒 Secure Access

Only you can view or edit your trade history through secure login.
